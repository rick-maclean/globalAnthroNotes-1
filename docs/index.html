    Paratext 8 Biblical Culture Notes Resource (BCNen):

    1. View in HTML (English) | Download latest English BCNen.p8z.

    Biblical Culture Notes Installation:

    1. Download Latest BCNen.p8z (select above, and click on “Download” icon) and Save or Copy to your “My Paratext 8 Projects\_Resources\” folder.
    2. In Paratext, File > Open File/Resource… to a scripture project or resource
    3. View > Show Consultant Notes > BCNen - Biblical Culture Notes (en).
    4. Navigate to Ruth 1:1 in the scripture project or resource
